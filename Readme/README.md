# NHANES Data Analysis in R

## Project Overview
This project performs exploratory data analysis (EDA) on a built-in NHANES dataset available in R. The analysis includes data cleaning, missing value handling, factor conversion, and visualization to understand health-related variables.

---

## Objective
- Understand structure of dataset
- Clean and preprocess data
- Handle missing values
- Convert variables to factors
- Perform exploratory data analysis
- Generate visualizations

---

## Tools and Technologies
- R (Base R)
- dplyr
- ggplot2

---

## Dataset Information
The dataset used in this project is a built-in dataset available in R for learning and analysis purposes. It contains health-related variables used for practicing data preprocessing and exploratory data analysis techniques.
